In Application Tutorial

After Downloading The Unity Application, the engine prompts you with a few templates to start your first game. Once you're in, It provides 5 tutorials to start the user off in Unity.

Walkthrough 1: Views/Windows

-There are two views in unity: Scene and hierarchy

Scene View: Allows you to navigate around and edit the scene while also showing the 3d/2d perspective depending on the project

Hierarchy View: Shows Everything in the current scene that is open in the "Scene View"

-If you select an object in the hierarchy view and press F in the scene view it will center the view on the selected object

-Game View is where you can test your game

-Project Window:

Displays all available assets that you can use in the project from 3d models, audio files, textures, and more

-Inspector Window:

You can change the properties or settings of whatever you have selected

Walkthrough 2: Changing Properties

-Component Properties allow game objects to do things (moving, jump height, etc.)

-These properties are changed in the Inspector Window

Walkthrough 3: Changing a Color

-Under the "Sprite Renderer" in the "Inspector View" you can select the "color" property to change the color of an object

Walkthrough 4: Adding a Prefab

Prefab: This is a game object, complete with all components and properties, that is stored in a asset. This is essentially a reusable preset that can be put at any point in the game

-These can be found in the "Project Window"

-Prefabs all have a "blue cube" icon next to them in the "Project Window"

-You can copy prefabs from the "Project Window" into the "Hierarchy Window" this will spawn it into the scene. (You can click on drag to copy them over)

-after framing the selected object (F) click on the "Move Tool" in the top left of the toolbar (Shortcut is "W"). You can then drag the object using the arrows (red= left/right) (green= up/down)

-There is also the "Rotate Tool" (E) and the "Scale Tool" (R)

Walkthrough 5: Build and Share

Build Settings Window:

-Shortcut is (ctrl+shift+B) or you go to the top right, click on file and select "Build Settings"

*Scene in Build Section: This is in the top half of the "Build Settings Window"

-This portion of the window will show all the scenes in your project that will be part of your "Build"

-If you want your game to work on a web browser you need to select WebGL after you have selected "Add Open Scenes"

-"Add Open Scenes" on the right where it says "Platform" is where you select what platform your game will run on

-After you have selected your platform hit the "Build" button at the bottom of the "Build Settings" window, and select the folder you wish to save your project in

*Now got to the "Window" button on the toolbar and select "Share WebGL Game"

-Finally, you will name your game, add a thumbnail, and upload it (When the upload is complete a link to your game will be given
